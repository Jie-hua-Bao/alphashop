# 這是 AC 學期三 React A3 作業

主要練習，以 React 方式將 HTML 結構拆分並組件起來

目前進度:未使用CSS Modules改寫，CSS暫時先引用教案
### 請遵循以下步驟

#### step 1
打開終端機 輸入指令 
 `git clone  'https://github.com/Jie-hua-Bao/alphashop.git'`

#### step 2
輸入指令 `cd alphashop`
進入專案資料夾

#### step 3
輸入指令 `npm install`
安裝所需套件

#### step 4
輸入指令 `npm start`
啟動專案

#### step 5
啟動成功 
網頁將在本地端
`http://localhost:3000`運行


### 環境建構
react 18.2.0
Node.js 14.18.1
npm 6.4.15
